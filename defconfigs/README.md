# CachyMod default configuration files

Do not edit the files in this directory.
Rather, make a copy to your configuration folder.

```bash
mkdir -p ~/.config/cachymod
cp 6.18/*.conf ~/.config/cachymod/
cp 6.19/*.conf ~/.config/cachymod/
```

Changes can be made with the `confmod.sh` utility.

Building a kernel is with the `build.sh confname` command,
as shown in the README.
